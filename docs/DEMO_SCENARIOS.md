# Сценарии для видеодемонстрации (скринкаст)

**Рекомендуемая длительность:** 5–8 минут  
**Инструменты записи:** OBS Studio, Windows Game Bar (Win+G), Loom

## Подготовка

```bash
# Терминал 1
npm run mock

# Терминал 2
npm run dev
```

Откройте http://localhost:5173

---

## Сценарий 1: Запуск и обзор дашборда (1 мин)

1. Показать карту с ~250 маркерами транспорта.
2. Указать индикатор «Онлайн» (SSE подключён).
3. Прокрутить KPI: ТС на линии, скорость, пунктуальность.
4. Отдалить карту (zoom out) — показать **кластеризацию** маркеров.
5. Приблизить кластер — spiderfy / раскрытие маркеров.

**Озвучка:** «Дашборд получает данные в реальном времени через SSE. При большом количестве ТС используется кластеризация Leaflet.»

---

## Сценарий 2: Чат-бот — трамваи на северо-западе (2 мин) ⭐

1. Открыть чат-ассистент (правый нижний угол).
2. Ввести: **«покажи трамваи на северо-западе»**
3. Показать:
   - Ответ бота: «Показано N трамваев на Северо-Запад»
   - Карта автоматически перемещается в северо-запад
   - Остаются только маркеры трамваев 🚊
   - Счётчик «ТС (Северо-Запад)» обновился
4. Кликнуть на маркер — popup с маршрутом, скоростью, задержкой.

**Озвучка:** «Ассистент распознаёт intent, применяет фильтры к карте и отвечает с точным количеством отфильтрованных ТС.»

---

## Сценарий 3: Поиск задержек (1 мин)

1. В чате: **«Где пробки?»** или **«Покажи задержки»**
2. Показать подсветку опаздывающих маркеров (красная рамка, badge «!»).
3. Ответ бота со списком ТС.

---

## Сценарий 4: Графики и оптимизация (1.5 мин)

1. Нажать «Загрузка за неделю» или написать **«Построй график загрузки за неделю»**
2. Показать линейный график в правой панели.
3. Быстро переключать период (если есть UI) — отметить плавность благодаря debounce.
4. Экспорт PNG / CSV.

---

## Сценарий 5: Production деплой (30 сек)

1. Показать Vercel URL (если задеплоено).
2. Показать `/health` mock-backend на Render.
3. Кратко: «Frontend на Vercel, API проксируется на Render.»

---

## Сценарий 6: Нагрузочный тест (30 сек, опционально)

```bash
npm run load-test
```

Показать вывод: REST P95, SSE metrics, PASS статусы.
