# 📋 Technical Specification: Transit Dashboard

**Версия:** 1.0  
**Стек:** Vue 3 + Pinia + Leaflet + Chart.js + Vite  
**Бэкенд:** Node.js/Express + Vercel Serverless

---

## 🎯 Видение продукта

Единая аналитическая платформа для мониторинга городского транспорта, где **диспетчер и аналитик общаются с системой на естественном языке**, а AI-ассистент динамически перестраивает дашборд под их задачи.

## 🎨 Функциональные требования

### 1. Карта (Leaflet)

- **Базовый слой:** OpenStreetMap / Mapbox
- **Маркеры ТС:** с анимацией движения (interpolation между точками)
- **Кластеризация:** при zoom out — группировка маркеров
- **Слои:**
  - Vehicles (ТС)
  - Routes (линии маршрутов)
  - Stops (остановки)
  - Heatmap (загруженность)
  - Incidents (инциденты)
- **Фильтры:** по типу ТС, маршруту, статусу
- **Popup:** детали при клике на маркер

### 2. Графики (Chart.js)

- **Типы:** Line, Bar, Doughnut, Scatter, Heatmap (через плагин)
- **Интерактивность:** zoom, pan, tooltip, legend toggle
- **Адаптивность:** responsive, mobile-friendly
- **Экспорт:** PNG, CSV

### 3. Панель метрик (KPI Cards)

- Количество ТС на линии
- Средняя скорость по городу
- % punctuality
- Пассажиропоток сегодня
- Активные инциденты
- Среднее время ожидания

### 4. AI-ассистент

- **Streaming ответов** (SSE)
- **Tool calling:** бот вызывает действия на дашборде
- **Confirmation flow** для деструктивных действий
- **Context-aware:** бот "видит" текущий фильтр, выбранный маршрут и т.д.
- **Quick actions:** кнопки быстрых запросов

## ⚡ Нефункциональные требования

| Категория | Требование | Метрика |
|-----------|-----------|---------|
| **Производительность** | Плавная анимация маркеров | 60 FPS при 500+ ТС |
| **Производительность** | TTFT (Time to First Token) | < 800ms |
| **Производительность** | Обновление карты | < 100ms на пакет |
| **Надёжность** | Uptime | ≥ 99.5% |
| **Масштабируемость** | Поддержка ТС | 10 000+ одновременно |
| **Доступность** | WCAG 2.2 AA | Lighthouse ≥ 95 |
| **Безопасность** | RBAC | Разделение ролей Диспетчер/Аналитик |

## 🚫 Out of Scope (MVP)

- Мобильные приложения
- Интеграция с реальными GPS-трекерами
- Платная подписка / биллинг
- Мультиязычность
- Голосовой ввод
- Прогнозная аналитика на ML
